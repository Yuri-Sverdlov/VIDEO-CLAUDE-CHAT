## Суть проекта

`VIDEO-CLAUDE-CHAT` — локальный Python-пайплайн, который из раскадровки на **6 сцен**
собирает **30-секундный** анимационный ролик-поздравление «Юная чемпионка» через API
[fal.ai](https://fal.ai).

**Пайплайн (3 стадии, стейдж-гейт):**
- **Стадия A** — text-to-image: 6 ключевых кадров (FLUX) → `output/frame_01..06.png`
- **Стадия B** — image-to-video: каждый кадр → ~5-сек клип (Wan) → `output/clip_01..06.mp4`
- **Стадия C** — ffmpeg склеивает 6 клипов → `output/champion_movie.mp4`

Раскадровка: `storyboard.svg`. Запуск: `README_RUN.md`. Код: `champion_movie.py`.
Исторический бриф: `MY-COMMENTS/architect-brief-original.md`.
